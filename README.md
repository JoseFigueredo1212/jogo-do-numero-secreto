# 🎮 JS Game - Projeto de Jogo em JavaScript

Bem-vindo ao **JS Game**! Este é um projeto simples de jogo desenvolvido em JavaScript, HTML e CSS, criado como parte do curso de JavaScript. 🚀

## 📖 Descrição

Este jogo é uma demonstração básica de conceitos de JavaScript, incluindo manipulação do DOM, eventos e lógica de jogo. Divirta-se jogando e explorando o código! 😄

## ✨ Funcionalidades

- 🎯 Jogabilidade interativa
- 🎨 Interface amigável com CSS
- 📱 Responsivo para diferentes dispositivos
- 🏆 Sistema de pontuação (se aplicável)

## 🚀 Como Executar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em seu navegador web.
3. Divirta-se jogando! 🎉

## 🛠️ Tecnologias Utilizadas

- **HTML5** 📄
- **CSS3** 🎨
- **JavaScript** 💻
